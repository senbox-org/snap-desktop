Import SRTM DEM

Imports a tile of the Shuttle Radar Topography Mission (SRTM) elevation model at 1 arc second resolution.

If a requested tile is not found in the SRTM Folder specified in the Settings, then it will download the tile from SNAP servers.

Since the SRTM height information is referred to geoid EGM96, not WGS84 ellipsoid, correction has been applied to obtain height relative to the WGS84 ellipsoid.
