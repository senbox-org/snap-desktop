Annotation Metadata Overlay

A more complete help documentation is TBD

File level metadata can be accessed with either of the following: 

    <GLOBAL_ATTRIBUTE={property_name}>
    <GLOBAL_ATTR={property_name}>

Band level metadata can be accessed with either of the following: 

    <BAND_ATTRIBUTE={property_name}>
    <BAND_ATTR={property_name}>

File level information properties are:

    <INFO=file>            - File name (as it appears in the File Manager)
    <INFO=file_location>   - File location (physical location from where file was read)
    <INFO=product_type>
    <INFO=scene_start_time>
    <INFO=scene_end_time>

Band level information properties are:

    <INFO=band>
    <INFO=units>
    <INFO=band_description>
    <INFO=scene_height>
    <INFO=scene_width>
    <INFO=scene_size>       - Combination of scene_width and scene_height
    <INFO=wavelength>
    <INFO=angle>
    <INFO=flag_coding>
    <INFO=valid_pixel_expression>
    <INFO=no_data_value>
    <INFO=is_no_data_value_set>
    <INFO=is_no_data_value_used>
    <INFO=is_scaling_applied>
    <INFO=scaling_factor>
    <INFO=scaling_offset>
    <INFO=is_log_scaled>
    <INFO=is_palette_log_scaled>
    <INFO=node_display_names>
    <INFO=node_names>
