Help Menu

The Sentinel Toolbox Help Menu contains:

  •   Help Topics
    Opens the Sentinel Toolbox help.
  • Context search
    Google search context
  • STEP Home Page
    Opens the STEP web site:
  • Report an issue
    Directs to a web page where it is explained how you can report issues:
  • Tutorials
    Open the tutorials page on the step web site:
  • Show Log Directory
    Opens the file browser with the location of the log files SNAP generates. This is useful to inspect the possible cause of errors and for issue reporting.
  • Check for Updates
    Checks if new updates are available for the installed modules.
  •   About SNAP
    Opens the 'About SNAP' window.
