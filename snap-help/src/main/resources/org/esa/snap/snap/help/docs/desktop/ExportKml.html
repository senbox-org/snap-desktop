Export KML

Export Displayed Image as KML

This command allows the export of the loaded image to a KML file. Only images from loaded bands, that are in lat/lon projection can be exported, reprojection can be then needed.

Following features of the product are exported:

  • raster data
  • colour legend
  • pins

The exported image is saved in the KML the Keyhole Markup Language (link: ). To view this file a version of Google Earth (link: ) is necessary.
