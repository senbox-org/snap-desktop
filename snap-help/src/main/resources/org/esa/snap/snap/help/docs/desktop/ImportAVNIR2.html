Import AVNIR-2

The AVNIR-2 reader enables the Sentinel Toolbox to import CEOS formatted AVNIR-2 Level-1 data products.

Note: The AVNIR-2 reader is implemented in line with the "ALOS AVNIR-2 Level 1 Product Format Descriptions Rev.G" () and is capable of reading data products of Level 1A, 1B1 and 1B2.
