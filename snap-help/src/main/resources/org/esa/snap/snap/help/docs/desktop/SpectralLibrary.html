The Spectral Library Tool

The Spectral Library Tool lets you extract spectral profiles from products (cursor / pins), preview them, and store them in spectral libraries for later reuse.
A click on the Selecting "Spectral Library" from the View --> Tool Windows menu opens the tool window.

The tool window is split into two main areas:

  • Library - manages spectral libraries and stored profiles (add/remove/rename, attributes, import/export).
  • Preview - shows extracted spectral profiles (e.g. from cursor position or pins).

Note: This page is currently a placeholder. Detailed workflow and screenshots will be added once the tool is finalized.

Library

Libraries store spectral profiles persistently. You can create a new library, rename or delete it, and add profiles from the preview.

Preview

Use the extraction actions to generate spectral profiles and inspect them in the preview chart. You can click a curve in the chart to select a profile (the selected profile is highlighted).
