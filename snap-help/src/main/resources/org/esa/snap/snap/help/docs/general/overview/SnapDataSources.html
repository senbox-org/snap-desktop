Data Access Details and Distribution Policy

  Instrument How to get data
1 MERIS

Data Access
MERIS data can be obtained by several means:

  • download of L1b and L2 RR data from 2002 to 2012 current from the MERCI catalogue
  • ordering L1b and L2 FR data from 2002 until current on CD/DVD from the EOLI-SA catalogue
  • download MERIS binned Level 3 data from the MERIS Level3 demonstration products page

MERCI includes all consolidated MERIS L1b and L2 RR data from 2002 to 2012 for download.

MERIS FR full swath data are processed on demand and delivered on media. They have to be selected in the EOLI-SA catalogue and ordered.

Data Policy
In order to download MERIS L1b and L2 data, a user account for the MERCI catalogue and/or the MERIS FTP server is required. Contact eohelp@esa.int.

MERIS Binned Level 3 data can be downloaded without any registration.

MERIS Binned Level-3

MERIS Binned Level 3 data can be downloaded without any registration.

2 (A)ATSR

Data Access

Archived data can be ordered through the EOLI-SA interface, a dedicated MERCI catalogue, or the NERC service.

Data Policy

In order to download (A)ATSR from MERCI, a user account for the MERCI catalogue is required. Contact eohelp@esa.int.

Ordering of (A)ATSR data from the EOLI-SA catalogue requires an accepted Cat-1 proposal. The proposal can be submitted from .

NERC requests registration before the data can be downloaded, contact: neodc@rl.ac.uk

9 MODIS

Data Access

MODIS data can be ordered through:

  • A dedicated MERCI catalouge
  • The Ocean Biology Processing Group Portal
  • The LP DAAC

Data Policy

  • No registration is required to download MODIS data from the OBPG portal.
  • Registration is required to download MODIS data from the MERCI catalogue, contact eohelp@esa.int
  • Registration is required to download MODIS data from the LP DAAC portal.
10 AVHRR/3

Data Access

AVHRR/3 data from NOAA are available through direct reception of the satellite data or via various http and ftp server.
AVHRR/3 data from Metop-A (Eumetsat) are available by the Eumetsat Earth Observation Portal.

Data Policy
Both ways of download require registration.
