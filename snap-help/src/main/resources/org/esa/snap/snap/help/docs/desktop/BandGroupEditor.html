Band Grouping Editor

Band Grouping Editor

This dialog allows you define, edit and manage Band Groups. A Band Group is a list of Bands of a product that can be grouped and given a group name. This functionality is similar to the auto-grouping entities provided by the ProductReader. User defined groups can be used in the Product Subset Dialog and in the SpectrumView to simplify handling of common band group definitions that are frequently used. The Band Group definitions are kept as a user-defined property and saved in a text file for easy manual editing of sharing. This file is stored at .snap/config/user_band_groups.json.

Manage Band Groupings

This panel allow to manage the user defined Band Groups. The drop-down list contains all Band Groups defined by the user. You can:

  • Add: create a new Band Group definitions. The Edit Band Grouping panel will be cleared and allows editing.
  • Remove: removes the currently selected Band Group definition from the list.
  • Store: saves the changes to the configuration file.

Edit Band Grouping

In this panel you can create and edit/modify Band Groups. To name a band group add the name in the appropriate text box. In the text-box Band Names, you can add the band names that shall be grouped. Names must be formatted either as one line, comma separated or as line separated band names.

Pressing OK adds the Band Group to the list, Pressing Cancel reverts the editing.
